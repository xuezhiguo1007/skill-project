# 北海涠洲岛旅游商品推荐 Skill

## 适用场景

- 用户需要北海、涠洲岛旅行装备、穿搭、防晒或出行用品推荐
- 用户希望根据海岛目的地特点挑选伴手礼或当地购买清单
- 用户给出了预算、人数、出行季节，希望整理成购买建议

## 工作要求

1. 必须先读取 `data/travel/shopping-guide.md`
2. 必须严格遵循 `recommendation-template.md` 中定义的结构顺序
3. 推荐时要优先解释适用场景和购买理由，而不是只列商品名
4. 必须按预算和优先级区分“必带/必买”和“可选”
5. 不要推荐资料中未出现的品牌、价格或商品类别

## 推荐原则

- 先匹配海边气候、上岛玩法和交通方式
- 再匹配同行人群，例如亲子、情侣、独自旅行
- 预算有限时，优先保留高频、刚需、复用性强的商品
- 如果商品更适合在出发前购买或到北海当地购买，要明确写出

## 输出前自检

- 是否包含推荐思路
- 是否包含必带或必买清单
- 是否包含可选升级项
- 是否包含购买提醒
- 是否使用中文 Markdown
